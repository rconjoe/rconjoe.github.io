How I Blog Without Leaving Vim

Vimwiki is a simple, powerful, open-source vim plugin that uses vim tricks and simple keybinds to make generating markdown much faster. Well, not actually markdown...but a custom vimwiki syntax very closely related to markdown. While you can use markdown (like the person writing this post), I chose to stick with the default syntax because I didn't want to have to set up any additional tooling to use Vimwiki's built in VimwikiAll2HTML feature.
